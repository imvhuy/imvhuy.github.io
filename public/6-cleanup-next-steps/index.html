Resource Cleanup and Next Steps

Resource Cleanup and Next Steps

Congratulations on building your complete serverless weather ETL pipeline! This final module focuses on properly cleaning up all AWS resources to avoid ongoing charges and explores opportunities for further enhancement.

Module Overview

After completing a hands-on workshop, it’s crucial to clean up resources to prevent unexpected charges. This module provides comprehensive cleanup procedures and suggests next steps for extending your weather pipeline.

Learning Objectives

By the end of this module, you will:

  • Understand the importance of resource cleanup in AWS
  • Execute comprehensive cleanup procedures for all created resources
  • Learn cost optimization strategies for future projects
  • Explore advanced features and improvements for your pipeline
  • Discover additional learning resources

Cleanup Importance

Proper cleanup is essential because:

  • Cost Management: Prevents unexpected charges from idle resources
  • Security: Removes unused permissions and access points
  • Best Practices: Establishes good AWS resource management habits
  • Account Hygiene: Keeps your AWS account organized and manageable

Complete Cleanup Checklist

6.1 QuickSight Resources

Priority: High (Ongoing subscription costs)

  1. Delete Dashboard

    • Remove your weather dashboard
    • Delete any shared versions

  2. Delete Datasets

    • Remove weather data datasets
    • Clear any cached data

  3. Cancel Subscription (if no longer needed)

    • Downgrade from Author to Reader
    • Cancel subscription to avoid monthly charges

6.2 Athena Resources

Priority: Medium (Query result storage)

  1. Drop Tables

    DROP TABLE weather_data_processed;
DROP TABLE weather_data_raw;

  2. Drop Database

    DROP DATABASE weather_analytics;

  3. Clean Query Results

    • Delete Athena query result files from S3

6.3 S3 Buckets and Data

Priority: High (Storage costs)

  1. Delete Raw Data Bucket

    aws s3 rm s3://weather-data-raw-[YOUR-ID] --recursive
aws s3 rb s3://weather-data-raw-[YOUR-ID]

  2. Delete Processed Data Bucket

    aws s3 rm s3://weather-data-processed-[YOUR-ID] --recursive
aws s3 rb s3://weather-data-processed-[YOUR-ID]

  3. Delete Athena Results Bucket

    aws s3 rm s3://aws-athena-query-results-[ACCOUNT-ID]-[REGION] --recursive

6.4 Lambda Functions

Priority: Medium (Minimal ongoing costs)

  1. Delete Weather Collector Function

    aws lambda delete-function --function-name weather-data-collector

  2. Delete Weather Processor Function

    aws lambda delete-function --function-name weather-data-processor

  3. Remove Function Triggers

    • Delete EventBridge rules
    • Remove S3 event notifications

6.5 CloudWatch Resources

Priority: Low (Free tier covers most usage)

  1. Delete Log Groups

    aws logs delete-log-group --log-group-name /aws/lambda/weather-data-collector
aws logs delete-log-group --log-group-name /aws/lambda/weather-data-processor

  2. Delete EventBridge Rules

    aws events delete-rule --name weather-collection-schedule

  3. Remove Custom Metrics (if any were created)

6.6 IAM Roles and Policies

Priority: Medium (Security and organization)

  1. Detach Policies from Roles

    aws iam detach-role-policy --role-name weather-lambda-role --policy-arn arn:aws:iam::aws:policy/AmazonS3FullAccess
aws iam detach-role-policy --role-name weather-lambda-role --policy-arn arn:aws:iam::aws:policy/CloudWatchLogsFullAccess

  2. Delete IAM Roles

    aws iam delete-role --role-name weather-lambda-role
aws iam delete-role --role-name quicksight-athena-role

  3. Remove Custom Policies (if created)

6.7 Systems Manager Parameters

Priority: Low (No ongoing costs)

  1. Delete Parameter Store Values 
    aws ssm delete-parameter --name /weather-etl/openweathermap/api-key

Final Cost Summary

After completing cleanup, your estimated total workshop cost should be:

  • Lambda Execution: ~$0.10 (within Free Tier)
  • S3 Storage: ~$0.20 (minimal data stored)
  • Athena Queries: ~$0.15 (small dataset queries)
  • QuickSight: ~$3-4 (if using free trial)
  • Data Transfer: ~$0.05 (minimal)

Total Workshop Cost: Under $5 (excluding QuickSight subscription)
